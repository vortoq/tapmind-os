# Contributing to TapMind OS

We welcome new apps, UI improvements, and bugfixes!

## To Contribute:

1. Fork the repo
2. Create your branch (`feature-xyz`)
3. Add new `.py` app to `/apps/`
4. Submit a pull request

Join the rebellion. Build something great.
